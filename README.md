# jumblesolver

>Solves Newspaper Jumbles

Example:

'''bash
Enter in you word: kaaill
Checking letters for length...
Finalizing List...

--> alkali
'''

## Build

'''bash
mkdir build && cd build
cmake ..
make
./jumblesolver
'''
